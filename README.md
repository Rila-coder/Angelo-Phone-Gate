# Angelo Mobile Gate

Angelo Mobile Gate is an **e-commerce website** for mobile-related products.  
It allows users to browse, search, and purchase mobile phones, accessories, and other electronics with a smooth and responsive interface.

## Features

- **Product Categories**: Mobile phones, chargers, earphones, power banks, and more.  
- **Product Details**: View detailed information about each product.  
- **Add to Cart**: Users can add products to their cart and manage them.  
- **User Authentication**: Users must log in to purchase, add to cart, or give feedback.  
- **Admin Panel**:  
  - Add, update, or delete products by category  
  - View customer inquiries and feedback  
  - Manage the website dynamically  
- **Special Offers**: Highlighted products with discounts.  
- **Responsive Design**: Works well on desktop, tablet, and mobile devices.  

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript  
- **Backend**: PHP  
- **Database**: MySQL
